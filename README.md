# Тетрис на Pygame


Простая реализация классической игры «Тетрис» на Python с использованием библиотеки Pygame.


## Описание

Игра включает:
- классическое управление тетромино (движение влево/вправо, падение, вращение);
- подсчёт очков за удалённые линии;
- повышение уровня сложности со временем;
- панель статистики (счёт, уровень, следующая фигура);
- экран «GAME OVER» с итоговыми результатами.


## Требования

- Python 3.7+
- Pygame 2.0+

Все зависимости перечислены в `requirements.txt`.

Управление

    ← — сдвиг влево;

    → — сдвиг вправо;

    ↓ — ускоренное падение;

Правила

    Собирайте полные горизонтальные линии, чтобы удалять их и набирать очки.

    За каждую удалённую линию — 100 очков.

    Уровень повышается каждые 500 очков, скорость падения фигур увеличивается.

    Игра заканчивается, если блоки достигают верхней границы экрана.

Структура проекта

    main.py — основной код игры;

    requirements.txt — список зависимостей;

    README.md — документация.

Настройки (константы в main.py)

    SCREEN_WIDTH, SCREEN_HEIGHT — размер окна;

    GRID_SIZE — размер ячейки сетки;

    COLUMNS, ROWS — количество столбцов и строк игрового поля;

    fall_speed — начальная скорость падения фигур (сек);

    COLORS — палитра цветов для фигур.
